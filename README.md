# Kasavilevari

A web application / future mobile app which enables people of Bengaluru to dump waste generated at their house, at a minimal cost and keep the city clean.

#Tech Stack 
1. Backend:  Django/Django REST framework
2. Database: SQLite (Development), PostgreSQL (Production)
3. Frontend: TBD
4. Infra:    TBD (AWS Planned)

### Getting Started (Local Development)

## Pre-requisites 

1. Python3 and above.
2. Git

## Setup 

--bash

- git clone https://github.com/Sharath787/Kasavilevari.git
- cd kasavilevari
- python -m venv venv
- source venv/bin/activate # Windows venv\Scripts\activate
- pip install requirements.txt
- python manage.py  runserver

## You can access the local site at

- http://127.0.0.1:8000/
